# 选择器

## 种类

1. 通配符选择其
```css
*{}
```
2. 标签选择器
```css
body{}
```
3. 类选择器
```css
.app{}
```
4. id选择器
```css
#app{}
```
5. 分组选择其
```css
.app,.bar,.foo{}
```
6. 后代选择器
```css
#app .foo{}
```
7. 交集元素选择其
```css
#app.foo{}
```
8. 父子元素选择器
```css
#app>.foo{}
```
9.  伪类选择器
```css
a:hover{}
```
10. 属性选择器
```css
[title='add']{}
```
11. 兄弟元素选择器
```css
span~p{}
span+p{}
```
